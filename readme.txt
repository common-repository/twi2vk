Upload '/twi2vk/' to the '/wp-content/plugins/' directory2. Activate the plugin through the 'Plugins' menu in WordPress3. Go to Settings page and fill necessary fields== Changelog === 0.1.1 =* FIX: Changed tweets source url.= 0.1 =* Public version
